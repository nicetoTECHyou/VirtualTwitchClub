Methode 1: VB-Audio Virtual Cable (EMPFOHLEN - kostenlos & zuverlässig)
  1. Von vb-audio.com herunterladen und installieren (CABLE Input)
  2. Windows Soundeinstellungen: Wiedergabe = deine Lautsprecher, Aufnahme = CABLE Input
  3. CABLE Input im Dropdown oben wählen & "Verbinden" klicken
  4. Fertig! Jetzt hört das Overlay was du hörst
Methode 2: Stereo Mix aktivieren (kostenlos, kein Extra-Software)
  1. Rechtsklick auf das Lautsprecher-Symbol in der Taskleiste → "Sounds"
  2. Tab "Aufnahme" → Rechtsklick → "Deaktivierte Geräte anzeigen"
  3. "Stereo Mix" oder "Stereomix" → Rechtsklick → "Aktivieren"
  4. Stereo Mix als Standard-Aufnahmegerät setzen
  5. Dann oben im Dropdown "Stereo Mix" wählen!
Methode 3: Desktop Audio Button (Bildschirmfreigabe)
  1. Button klicken → Bildschirmfreigabe-Dialog öffnet sich
  2. WICHTIG: Häkchen bei "Audio teilen" / "Share system audio" setzen!
  3. Beliebigen Bildschirm wählen → Teilen
  4. Hinweis: Funktioniert nicht in allen Browsern zuverlässig
