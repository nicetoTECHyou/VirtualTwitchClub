Ersteinrichtung - Los geht's!

1. System-Check

Pruefe Node.js Installation...

Node.js wird benoetigt um die Server zu betreiben.
Klicke auf "Herunterladen" um Node.js zu installieren.

2. Twitch Kanal

Wenn leer, wird der Kanalname verwendet.

3. Twitch OAuth Token

Token hier generieren (Benutzername: dein Bot-Name, Scopes: chat:read, chat:write)

4. Fertig!

Alles ist eingerichtet. Die Services werden jetzt gestartet.
Du kannst jederzeit ueber die Tabs auf die Admin-Panels zugreifen.

Backup & Restore

Komplettes Backup aller Einstellungen aus allen 3 Apps + Launcher. Inklusive Blacklist, Whitelist, Chat-Commands, Scenes und Twitch-Credentials.

Hinweis: Medien-Dateien (Sounds, Videos) werden nicht exportiert - nur die Einstellungen. Nach einem Import muss die App neu gestartet werden damit alle Aenderungen wirksam werden.

OBS Setup Guide

Folge diesen Schritten um die Overlays in OBS Studio hinzuzufuegen.

OBS Overlay URLs

Schritt-fuer-Schritt Anleitung

1
Oeffne OBS Studio
2
Klicke unter "Quellen" auf das + Symbol
3
Waehle "Browser" und klicke auf OK
4
Gib einen Namen ein (z.B. "Club Dancers") und klicke auf OK
5
Trage die URL ein: http://localhost:3333
6
Setze Breite: 1920 und Hoehe: 1080
7
Wiederhole die Schritte 2-6 fuer alle 3 Overlays (siehe URLs oben)

Tipp

Die Reihenfolge der Browser-Quellen in OBS bestimmt, welches Overlay im Vordergrund steht. Zieh die Quellen einfach um sie neu anzuordnen.
